How strongly the image follows the prompt. Higher = tighter adherence but can over-saturate / harden contrast; lower = softer and more natural. Default 7.
Advanced parameters
CFG override applies a different guidance value during only part of the run. Start/End % are points on the denoising timeline (0 = first step, 1 = last); the override CFG is used only inside that window. The default (3 over 0.90–1.00) lowers guidance for the final 10% of steps to reduce over-sharpening and color burn. Set override = Guidance, or start ≥ end, to disable. The % is a fraction of the step count, so it covers fewer steps in Turbo (12) than Quality (48). Batch size = how many images per render (more = more VRAM and time).
